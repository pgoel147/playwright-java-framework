# Playwright Java Automation Framework

A professional test automation framework built with Java, 
TestNG and Maven.

## Tech Stack
- Java 17
- TestNG 7.8.0
- Maven 3.9.11
- Git / GitHub Actions (CI/CD - coming soon)

## Project Structure
src/test/java/com/prashant/automation/tests/
└── MyFirstTest.java    # First TestNG test suite

## Tests Included
| Test Name       | Description              | Status |
|----------------|--------------------------|--------|
| verifyAddition  | Validates basic addition | PASS   |
| verifyMyName    | Validates string assert  | PASS   |

## How to Run
Clone the repo:
git clone https://github.com/pgoel147/playwright-java-framework.git

Run all tests:
mvn test
